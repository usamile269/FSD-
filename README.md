Ahmad Mini Bot 🤖

A powerful WhatsApp Multi-Device bot with advanced features and commands.

Features

- 600+ Commands
- Auto Reply System
- Downloader Commands
- Group Management
- Fun & Games Commands
- Media Tools
- Admin Panel
- Easy Configuration
- Multi-Device Support

Installation


cd Ahmad-Mini-Bot
npm install
npm start

Requirements

- Node.js v18 or higher
- npm
- Stable Internet Connection

Configuration

Edit the configuration files and add your required settings before starting the bot.
